# Тестовый Документ

Это тестовый документ для проверки правильности работы нашей системы перевода.

## Возможности

- Автоматический перевод
- Поддержка нескольких языков
- Сохранение кода

```javascript
// This code should NOT be translated
function hello() {
  return 'Hello World!'
}
```

Готов к тестированию!
